# Visual DNA

Use this file to lock the aesthetic before composing layouts or code.

## Core signal stack
Prioritize these signals in order:
1. grid rigor
2. typographic hierarchy
3. asymmetrical balance
4. scale contrast
5. restrained palette
6. modular geometry or cropped imagery
7. optional texture or distortion

If the first four signals are weak, the design will not read as swiss-editorial even if the palette and shapes are correct.

## Mode matrix

### 1. Classic swiss
Use for the purest interpretation.

Signals:
- strict grid
- restrained black, white, red palette
- clean sans typography
- flat fields and rules
- minimal image treatment
- almost no texture

Use when:
- the user wants timeless, rational, typographic design
- the work must feel disciplined and authoritative
- the content itself is strong enough to carry the composition

Avoid:
- distortion
- excessive layering
- noisy textures
- more than one accent color

### 2. Contemporary editorial
Use for most landing pages, websites, and presentation systems.

Signals:
- classic swiss foundation adapted for digital canvases
- bigger type and more dramatic cropping
- more negative space
- occasional modular rounded geometry
- stronger use of image blocks or abstract forms
- still very restrained, but not museum-pure

Use when:
- the task is a modern product or brand website
- the result should feel design-forward but not chaotic
- the work needs to read well as code on a large desktop canvas

Watch for:
- slipping into generic modern minimalism
- soft luxury aesthetics
- too many perfectly centered components

### 3. Experimental poster
Use when the brief supports tension and a more expressive composition.

Signals:
- same grid discipline underneath
- warped lines, overprints, giant numerals, noise, or harder abstraction
- stronger crop decisions
- more dramatic black and red fields
- visible texture or photocopy influence when requested

Use when:
- the brand can support art-school or poster energy
- the user wants a bolder, more graphic landing page or deck
- the concept benefits from abstraction or controlled friction

Watch for:
- chaos replacing discipline
- unreadable hierarchy
- textures taking over the design

## Layout signatures from the references
Use these recurring signatures:
- large numeral or index marker paired with a compact text block
- oversized headline occupying multiple grid cells
- small metadata placed in the margin or near a structural edge
- deliberate empty columns or rows
- hard geometric shapes aligned to the same system as the type
- black, off-white, and red or orange as the default emotional register
- poster logic applied to web sections and slides

## Typographic character
Aim for:
- neutral grotesk clarity
- confidence rather than friendliness
- tight, decisive display settings
- small sharp metadata blocks
- occasional line breaks that create abstract shape

Do not aim for:
- playful tech branding
- quirky startup friendliness
- decorative serif editorial fashion styling
- bubbly rounded UI typography

## Shape vocabulary
Prefer:
- circles
- semicircles
- quarter-circles
- rounded rectangles
- straight rules
- bars
- repeated line fields
- spirals or arcs only when they feel measured

Avoid:
- random abstract blobs
- loose illustration-style doodles
- soft liquid gradients

## Texture vocabulary
Optional only.

Safe textures:
- grain
- copier noise
- paper tooth
- halftone
- faint scan artifacts

Unsafe textures:
- heavy grunge everywhere
- glitch effects with no conceptual reason
- texture that reduces contrast or muddies small text

## Copy vocabulary
Make copy feel editorial and factual.

Good moves:
- concise declarative headlines
- issue numbers, edition numbers, dates, locations, version labels
- short descriptive paragraphs
- annotations and captions
- side notes and credits

Bad moves:
- long persuasive marketing paragraphs in every section
- excessive exclamation or hype language
- sentimental brand slogans with no typographic backbone
